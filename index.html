About This Project

This interactive platform provides insights into Pune's real estate market by visualizing property data on a map and offering price predictions based on property features.

Key Features
  • Interactive map with clustering for easy navigation
  • Filter properties by area, BHK, furnishing, and price
  • Machine learning-based price prediction
  • Detailed property information on click
  • Responsive design for all devices
Data Source

The property data includes information about area, locality, BHK configuration, furnishing status, distances to key amenities, and historical prices across Pune.
